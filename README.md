# Is Even
A golang package to check is a number is even.

## Installation
```bash
go get github.com/AvicennaJr/IsEven
```
## How To Use
```go
package main

import (
	"fmt"

	iseven "github.com/AvicennaJr/isEven"
)

func main() {
	n := 4
	if iseven.IsEven(n) {
		fmt.Println("It is even")
	} else {
		fmt.Println("It is not even")
	}
}
```
